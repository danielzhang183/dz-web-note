## Primitive Type

### Javascript

- string
- number
- bool
- null
- undefined
- object
- array
- symbol
- bigint

### Typescript

- any, can represent any kind of values
- unknown, 有点类似于 any，但是更安全，因为对 unknown 类型的值做任何事情都是不合法的：
- never, values that never occur
- enum
- tuple, a fixed length array where each element has a particular type
- void
